Dirección:

Busca una ciudad o dirección y se mostrará una marca con su longitud y latitud. Ejemplo: 'Calle del Barquillo, Madrid'. También, para añadir una marca, puedes hacer doble click en cualquier punto del mapa.
Si añades varias direcciones se ajustará el zoom del mapa y se centrará para que se visualicen todas las marcas
(Si pones todas las palabras 'Calle de...' o 'Paseo de la...' funciona muy bien)

Para la geolocalización utiliza el servicio Openrouteservice.org
